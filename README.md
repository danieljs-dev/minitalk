# How to use it ?
- Clone the repository:
```shell
git clone https://github.com/danieljs-dev/minitalk.git
```
```shell
cd minitalk
```
```bash
make
```
```shell
./server
```
- Open other terminal and run:
```bash
chmod +x get_server_pid.sh
source ./get_server_pid.sh
```
```shell
./client $SERVER_PID "Hello world"
```
